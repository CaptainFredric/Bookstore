FAQ

Quick answers about the catalog and how it’s built.

Built with HTML/CSS/JS
Is there a cart or checkout?

Not yet. This build focuses on browsing (search, filter, sort) and clean presentation. A cart and checkout can be added later.

Where do cover images come from?

Covers load from Open Library using ISBNs, with a clean fallback if a cover is missing.

Can I customize the books?

Yes. Update the books array in the script (title, author, genre, format, rating, ISBN, and optional price fields).
